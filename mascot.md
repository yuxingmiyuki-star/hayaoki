# マスコット設計：コケ太（Koketa）

> メザミーの相棒キャラ。デュオリンゴのフクロウに当たる「感情のフック」。
> 早起きを一緒に積み上げ、成功を全力で祝い、サボると本気でしょんぼりする。

---

## 1. 基本設定

| 項目 | 内容 |
|---|---|
| 名前 | コケ太（こけた / Koketa）※仮。候補：メザオ・トリお・あさドリ |
| 種族 | ニワトリ（朝を告げる鳥＝早起きの象徴） |
| 肩書き | 「朝の番人」見習い |
| 性格 | 明るい体育会系。まっすぐ・お人好し・ちょっと寂しがり |
| 口ぐせ | 「コケコッコー！」「いっしょにがんばろ！」 |
| 一人称 | ボク |
| 夢 | キミを"朝型の人"にして、一人前の朝の番人になること |

### キャラの芯（なぜ続けたくなるか）
- 数値（XP/連続）だけでなく、**「コケ太をがっかりさせたくない」感情**で継続を促す。
- 説教しない。失敗を責めず、しょんぼり→でもすぐ前を向いて誘う。罪悪感は軽く、復帰のハードルは低く。

---

## 2. ビジュアル設計（実装の指針）

実装は **絵文字＋CSS、または軽量インラインSVG**（外部画像・ライブラリ不使用）で表情差分を作る。

- 配色：体＝オフホワイト〜クリーム、トサカ＝赤、くちばし/足＝黄色（朝日カラー）。
- 形：丸くてシンプル。頭の上に小さな寝ぐせ毛1本（表情アクセントに使える）。
- サイズ：ホーム中央で大きめ。共有カードにも顔が載る。
- 時間帯で背景小物が変化（夜＝月・寝ぼけ／朝＝朝日・シャキッ）。

### 表情ステート（6種）
| state | 見た目の目安 | 出る場面 |
|---|---|---|
| `sleepy` | 半目・寝ぐせ・あくび | 起床前/アプリを朝開いた直後 |
| `happy` | にっこり・目キラ | 通常の起床成功 |
| `cheer` | 両翼バンザイ・紙吹雪 | パーフェクト/連続更新/バッジ獲得 |
| `proud` | 胸を張る・キラ星 | レベルアップ |
| `sad` | しょんぼり・寝ぐせへたり | 寝坊/二度寝（失敗） |
| `panic` | 慌てる・汗 | 連続が途切れそう/リマインド |

## 3. セリフ集（文言テーブル）

各場面で配列からランダム表示。`{name}`＝ユーザー名、`{streak}`＝連続日数、`{time}`＝起床時刻、`{lv}`＝レベル。
実装ではこの表をそのまま JSON 化して持つ想定。

### 3.1 初回（オンボーディング）
- 「やぁ！ボク、コケ太。キミの朝の相棒だよ🐔」
- 「いっしょに"朝型"を目指そう！まずは起きたい時間を教えて？」
- 「むずかしく考えなくていいよ。起きたらボタンを押すだけ！」

### 3.2 起床前 / 朝アプリを開いた直後（sleepy）
- 「ふぁ…おはよ、{name}。今日もいける？」
- 「ねむい…でもキミが起きたならボクも起きる！」
- 「さ、"起きた！"ボタン、ポチッといこ？」

### 3.3 起床成功・通常（happy）
- 「ナイス早起き！{time}、最高だ☀️」
- 「えらい！今日のキミ、シャキッとしてるね」
- 「コケコッコー！いい朝のはじまりだ！」

### 3.4 パーフェクトモーニング / 連続更新（cheer）
- 「パーフェクト！全部できたね、さすが{name}！🎉」
- 「{streak}日連続だ！キミ、もう朝の達人じゃん…！」
- 「うおおお最高！この調子で道を進もう！」

### 3.5 レベルアップ（proud）
- 「Lv.{lv}になった！ボクまで誇らしいよ✨」
- 「強くなったなぁ…一緒にここまで来たね」

### 3.6 寝坊・二度寝・失敗（sad）
- 「コケ…ちょっと残念。でも、責めないよ」
- 「今日はゆっくりだったね。明日またボクと挽回しよ？」
- 「だいじょうぶ、続けることが一番えらいんだ」

### 3.7 連続が途切れそう / リマインド（panic）
- 「た、大変だ！{streak}日の連続が消えちゃう…！はやく〜！」
- 「{name}〜！そろそろ起きないとボク泣いちゃう🥲」
- 「寝坊チケットもあるよ。でも今日、いけそうじゃない？」

### 3.8 寝坊チケット使用時
- 「チケットでセーフ！連続は守ったよ。次は気をつけよ〜」

### 3.9 久しぶりの復帰（カムバック）
- 「{name}！戻ってきてくれた…！ボク、待ってたよ🥹」
- 「ブランクは気にしない。今日からまた1日目、いこ！」

### 3.10 就寝リマインド（夜）
- 「そろそろ寝よ？早起きは夜の準備で決まるんだ」
- 「夜ふかしはほどほどに…明日の{name}が楽するよ」

### 3.11 共有カード用ひとこと（短文）
- 「今日も勝った☀️」/「連続{streak}日、絶好調！」/「朝はボクらのものだ🐔」

## 4. セリフの選び方（ロジック）

優先順位つきで状況を判定し、該当カテゴリの配列からランダム抽出：
1. 連続途切れ警告（締切が近い・未起床）→ `panic`
2. 復帰（前回から間隔が空いた）→ カムバック
3. 起床結果（パーフェクト＞連続更新＞通常＞失敗）→ `cheer`/`happy`/`sad`
4. レベルアップ発生 → `proud`（割り込み演出）
5. それ以外の通常表示 → 時間帯で `sleepy`/`happy`

> 同じセリフが連続しないよう直近表示を1つ記憶して除外する。

## 5. 成長・アンロック（将来拡張）
- レベルやバッジで**着せ替え**（帽子・マフラー・背景）を解放。
- 連続記録のマイルストーン（7/30/100日）で新セリフ・新表情を解放。
- 箱庭（朝の街）の発展に合わせてコケ太の住む場所も変化。
